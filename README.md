# 12kl_Garigas_veselibas_centrs_Gubrienko
Garigas veselibas centrs ir vienkāršs un vizuāli mierīgs labsajūtas rīks jauniešiem, kurā pieejami elpošanas vingrinājumi, dienasgrāmata un atgādinājumi ikdienas līdzsvaram. Projekts izstrādāts kā HTML/CSS prototips ar skaidru navigāciju un pārskatāmu izkārtojumu desktop skatam.
