# Garīgās veselības centrs — HTML/CSS/JS prototips

## Kā palaist
- Atver `index.html` pārlūkā (vai izmanto Live Server VS Code).
- Visas lapas ir savstarpēji sasaistītas caur navigāciju.

## Kas ir izdarīts (NPD kritērijiem)
- Semantiska HTML struktūra (`header`, `nav`, `main`, `section`, `article`, `footer`).
- Dizains un krāsu palete saskaņota ar maketu/specifikāciju.
- Responsīvs izkārtojums (Grid/Flex + media queries, mobilā navigācija).
- 5 lapas: sākumlapa, elpošana, dienasgrāmata, atgādinājumi, par mums.
- Funkcionalitāte (interaktīvie elementi):
  1) Elpošanas taimeris ar animāciju (4-7-8, kvadrāts, dziļā elpošana).
  2) Dienasgrāmata ar kalendāru + emocija + piezīmes (localStorage).
  3) Atgādinājumu slēdži (localStorage) + toast paziņojumi.

## Autortiesības
- Projekts izmanto Google Fonts (Montserrat, Roboto).
- `assets/hero.png` un `assets/team.png` ir **pašģenerēti** vietturis attēli.
